Computing and IT Project

The problem is:

Analysing network traffic data can be a complex task, especially for those who have no prior skills in networking technologies. There exist tools for this purpose such as Wireshark, but they are aimed at professional skilled networking experts.

There exists a lack of user-friendly tools and interfaces for visualising network traffic. This means there is a gap in understanding network performance and security for both technical and non-technical users.

The stakeholders (who will benefit from the project):

Network administrators, educational institutes, small businesses, cybersecurity professionals.

The user(s) (who will use the project) is/are:

Students and teachers in networking and cybersecurity education, IT professionals, individual users.

The scope of the problem is:

This project aims to address the lack of accessible tools for visualizing network data. By capturing data, processing it, and then representing it visually, this project should:

  • Simplify network data analysis
  • Enable data visualisation
  • Provide a cross-platform web-based tool that can be accessed from any device
  • Serve student, teacher, and professional networking needs

Research already done into problem, any existing software in the area:

Many users are locked out of reach of existing tools due to cost, learning curve, and availability.

There is a lack tools that are readily accessible through a web-based interface.

There is a lack of easy to use tools that simplify traffic analysis for users.

Wireshark: A well known tool for capturing and analysing network packets which has a learning curve and lacks visualisation.

Splunk: This offers data analysis and visualisation but is aimed at enterprise and therefore inaccessible.
