Degree Project: PCAPNG Network Analysis Platform

I developed a comprehensive full-stack web application that enables network students and professionals to upload, analyse, and visualise PCAPNG (Packet Capture Next Generation) files. This platform combines modern web technologies with advanced network analysis capabilities, providing an intuitive interface for complex packet inspection tasks.

The project demonstrates ability in full-stack development, network protocols, security implementation, and user experience design. It serves as a practical tool for network education while demonstrating enterprise-level software architecture and development practices.

Technical Architecture & Implementation

Backend Development (Django REST Framework)

  • Architected a robust REST API using Django with comprehensive JWT authentication system
  • Designed and implemented a sophisticated PcapngFile model with extensive metadata tracking including packet statistics, protocol distributions, IP traffic analysis, and temporal data
  • Integrated Scapy library for deep packet inspection, enabling analysis of TCP, UDP, ICMP, and ARP protocols with automatic extraction of network statistics
  • Implemented secure file upload system with validation, verification, and organized storage
  • Developed comprehensive error handling, security logging, and rate limiting to protect against malicious attacks

Frontend Development (React with Vite)

  • Built responsive single-page application using React with modern hooks and functional components
  • Implemented drag-and-drop file upload interface with real-time progress tracking and visual feedback
  • Created interactive data visualizations using Chart.js for protocol distributions, packet size histograms, and network topology displays
  • Developed centralized authentication context with secure JWT token management and automatic refresh capabilities
  • Designed accessible user interface following WCAG guidelines with proper ARIA labels and keyboard navigation

Security & Performance Features

  • Implemented JWT-based authentication with refresh tokens (60-minute access, 7-day refresh lifespan)
  • Built comprehensive input validation and sanitization to prevent injection attacks
  • Developed secure token storage utilities with automatic cleanup and session management
  • Optimized database queries for efficient handling of large PCAPNG files and metadata
  • Implemented background processing for CPU-intensive packet analysis operations
